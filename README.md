##  **SOBRE O PROJETO**

O **ENERGY PREDATOR** é um site criado com o objetivo de simular a venda de um energético, destacando:

* 🔹 **IDENTIDADE VISUAL FORTE**
* 🔹 **ESTILO MODERNO E CHAMATIVO**
* 🔹 **EXPERIÊNCIA SIMPLES PARA O USUÁRIO**
